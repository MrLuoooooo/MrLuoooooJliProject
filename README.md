# MrLuoooooJliProject

Monorepo containing two Go projects:

- **im-server/** — [JuggleIM](https://github.com/juggleim/im-server), an open-source high-performance instant messaging server based on the Actor model. Provides WebSocket connections, message distribution, group chats, push notifications, RTC, and 19 micro-services.

- **community/** — Community forum backend. Built with Gin, GORM, Uber Fx dependency injection, and SiliconFlow AI integration. Features user management, posts, comments, tags, follows, private messaging, and AI-powered search.
